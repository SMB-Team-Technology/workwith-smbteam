AD SPEND CALCULATION
======================================================================

Conservative ad spend: $12,000/mo
  - MVA/Car Accident row: Google PPC min $10,000 + LSA min $2,000 = $12,000
  - No Meta included: the transcript's stated goal is to optimize existing PPC and LSA
    channels for quality, not to expand into new channels at this time.

Aggressive ad spend: $50,000/mo
  - 20% rule: $3,000,000 (stated 2026 revenue goal) x 20% / 12 = $50,000/mo
  - Geographic market multiplier: Wilmington, DE is a mid-size market — Tier 4
    (significant regional), multiplier 1.0x. $50,000 x 1.0 = $50,000.
  - No Spanish campaign needed (not stated in transcript) — no additional multiplier.
  - Reverse math validation: ($3,000,000 / 12) / $7,500 avg case value = 33.3 target
    cases/month -> / 50% close rate = 66.7 required leads/month -> x ~$470 blended CPL
    (unweighted average, used only for this cross-check) = ~$31,000/mo. This is lower
    than the 20%-rule figure of $50,000, so per the rule ("use whichever of 20% rule or
    reverse math is higher") the $50,000 figure is used.
  - Tier cap check: Dominate's ad spend cap is $75,000/mo. $50,000 fits comfortably
    under this cap — no tier escalation needed for ad spend alone.

Average case value: not stated as a dollar figure in the transcript (only
revenue-per-case-day metrics were given: $40.22 litigation vs. $34.33 pre-litigation).
Using the MVA/Car Accident practice-area default midpoint of $7,500 per the
audit-write.md default table. This is a disclosed default, not a transcript-confirmed
figure, and is labeled as such in both the audit and the Sales Companion.

Close rate for ROI math: transcript's stated current rate, ~50% (target is 80% but not
yet achieved). Current rate is used for conservative, defensible ROI math in Block 4;
the 80% target is referenced qualitatively elsewhere in the audit (e.g. Growth Health,
Intake sections) as upside, not built into the Block 4 math itself.

Blended CPL (Accident & Injury, weighted ~83% Google Search / 17% LSA, reflecting the
firm's actual channel mix and spend proportions):
  Google Search mid = ($630+$780)/2 = $705
  LSA mid = ($210+$260)/2 = $235
  Blended = 0.83 x $705 + 0.17 x $235 = $585.15 + $39.95 = ~$625

ROI Conservative ($12,000/mo spend):
  Blended CPL $625 + 20% cushion = $750
  Leads = $12,000 / $750 = 16/mo
  Cases = 16 x 50% = 8/mo
  Revenue = 8 x $7,500 = $60,000/mo
  Return = $60,000 / $12,000 = 5.0x

ROI Aggressive ($50,000/mo spend):
  Blended CPL $625, no cushion
  Leads = $50,000 / $625 = 80/mo
  Cases = 80 x 50% = 40/mo
  Revenue = 40 x $7,500 = $300,000/mo
  Return = $300,000 / $50,000 = 6.0x

All ROI figures are estimates and are labeled as such in both the audit HTML and the
Sales Companion PDF.
